# Essentials Suite — A Collection of Durable, Minimalist Android Apps

Essentials Suite is an ecosystem of Android applications built with a simple philosophy:  
**software should be reliable, lightweight, private, and designed to last.**

No ads.  
No tracking.  
No unnecessary permissions.  
No feature bloat.  
Just tools that work — today, tomorrow, and years from now.

---

## 🌱 Project Philosophy

This project is inspired by the idea of “Toyota-style software”:  
long-lasting, efficient, easy to maintain, and focused solely on its purpose.

Each app in this ecosystem follows these principles:

- **Minimalism** — Only essential features, nothing more.
- **Durability** — Code that remains maintainable for years.
- **Privacy** — No analytics, no data collection, no third‑party trackers.
- **Lightweight** — Small APKs, low memory usage, fast startup.
- **Repairability** — Clean architecture, simple dependencies, open source.
- **Respect** — No ads, no dark patterns, no engagement traps.
